### Tugas
Buatlah program yang menterjemahkan angka romawi ke angka desimal.
```
Symbol       Value
I             1
V             5
X             10
L             50
C             100
D             500
M             1000
```

#### Example:
```
Input: s = "III"
Output: 3
Explanation: III = 3.

Input: s = "LVIII"
Output: 58
Explanation: L = 50, V= 5, III = 3.
```
